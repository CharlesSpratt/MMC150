Welcome to MMC150 - JavaScript & jQuery at CCAC!

This course introduces stduents to web application programming with JavaScript and jQuery library. Students create, test and debug scripts that include object methods and properties, data types, data selectsions and repetition structures, as well as window, form, frame and document objects.

Learning Outcomes:

Upon successful completion of the course, the student will:

  • Apply the fundamentals of the JavaScript Language
  • Create scripts, dialog boxes, confirm boxes, prompt boxes, framesets and frames.
  • Examine different types of variables, operators and data types.
  • Apply logical and conditional operators, functions and objects.
  • Display default and custom status bar messages.
  • Examine the porperties of document objects, form objects, string objects, date and math objects.
  • Correct data entry errors, detect browers and platforms and validate forms
